DATA & FILE OVERVIEW

Folder list:

Amorphous_B2O3_configuraions: Collection of 500 frames using a procedure starting from artificial B2O3 crystal with 100% boroxol.

Train-data : DFT data (training and validation) in raw (.raw) and binary file (.npy) formats.

DP_models : 4 independent DP models for 6 and 9 angstrom cutoff (frozen .pb files) and one MACE model.

Template-input-files: Sample input files for DP training, MD with LAMMPS, CP2K single point energy calculations
